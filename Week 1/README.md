Objective: Concatenate the monthly MLS data into one CSV

Taking monthly MLS data from a folder in CSV format as input, Concatenate.py transforms it into dataframes and cleans it by preferring _filled versions in duplicate scenarios by using a dictionary, and returns the dataframes in numerical order using regular expressions. Then, it takes out the extra columns of "latfilled" and "lonfilled" that are exclusive to the _filled versions. Afterwards, it filters and only keeps the rows with PropertyType of "Residential", and produces the fully concatenated in ascending order as a CSV. Includes row count before and after concatenation and Residential filter.
